# Automação WhatsApp com n8n

Projeto prático desenvolvido em n8n com foco em automação e lógica condicional.

# O que esse fluxo faz
- Recebe mensagens via Webhook
- Processa a mensagem com lógica IF
- usa o edit fields para filtrar dados que entram
- usa planilha integrada com google sheets para salvar nome e numero de quem mandou menssagem
- usa simple memory para lembrar de perguntas feitas pelo usuario
- Alimente com a groq.ia 
- Retorna resposta automática no WhatsApp
- Usado para testes de conversa consigo mesmo

# Ferramentas utilizadas
- n8n
- Webhook
- IF Node
- edit fields
- google sheets
- agentes de IA
- simple memory 
- Integração com WhatsApp

Projeto criado para fins de estudo e demonstração prática.
